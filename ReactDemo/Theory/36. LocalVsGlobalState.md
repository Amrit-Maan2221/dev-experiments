    </AuthContext.Provider>
  );
}

## 3ï¸âƒ£ Key Differences (Interview Friendly)

| Feature     | Local State              | Global State            |
| ----------- | ------------------------ | ----------------------- |
| Scope       | One component            | Whole app               |
| Managed by  | `useState`, `useReducer` | Context, Redux, Zustand |
| Performance | Faster                   | Can cause re-renders    |
| Complexity  | Simple                   | More complex            |
| Use case    | UI-specific              | Shared app data         |
